# Tachomon OTA Firmware

Over-The-Air (OTA) firmware updates for Tachmon water meter tacho output counter.

## Description
Tachmon is used for Water meter tacho output counter. This repository contains firmware releases for ESP32-based Tachmon devices.

## Latest Release
**Version**: v1.8.1

## OTA Update URL
Use the following URL in your ESP32 OTA configuration:
```
https://github.com/barryingham/tachomon_OTA_firmware/releases/download/v1.8.1/firmware.bin
```

## Installation
1. Download the firmware.bin file from the latest release
2. Configure your ESP32 with the OTA URL above
3. Trigger the OTA update on your device
